# 使用方法（放开注释同下）

Ctrl+Shift+/

目前改插件只支持 vscode，后续会开发 WebStorm 版本

# 修改按键快捷方式

打开 vscode，找到左上角文件 -> 首选项 -> 键盘快捷方式，输入 HTML-Comment，可以查看并修改按键。

# 支持列表

适用于 htm、html、asp、cfm、jsx、md、njk、php、svelte、svg、tsx、twig、vue、xml、xsl 等格式文件中任意 <!-- <tag></tag> --> 风格的代码块。
