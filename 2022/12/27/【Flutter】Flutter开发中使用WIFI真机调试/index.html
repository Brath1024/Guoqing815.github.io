# 使用 vscode 开发 flutter，用 wifi 或者 手机热点进行 无线调试过程

# 1. 手机连接 wifi 后在设置中找到 ip 和端口

# 2. 在 vscode 终端输入 adb connect 手机 IP: 手机端口
