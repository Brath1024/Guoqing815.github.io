# 宝塔部署 Docker—Mysql 镜像

如何在面板上使用 Docker 项目管理器快速创建多个版本的 MySQL？手把手的教你如何创建多个版本的 MySQL 服务
环境介绍：
宿主机：CentOS7.9
配置：2 核 4G（测试机器，生产环境建议配置高点）
面板版本：7.9.26（测试版）
Docker 项目管理器：3.9

Docker 版本：Docker version 20.10.14, build a224086
测试 MySQL 版本：
MySQL5.7.37
MySQL8.0.28
