# Linux 下使用 war 包部署 Jenkins

# 前言

jenkins 可以多种方式安装，可以 docker，也可以直接下载 war 包，然后 java -jar 方式启动。

# 备注:

  1. –httpPort 自定义端口
  2. jenkins 默认工作目录：/root/.jenkins
  3. nohup 后台启动
  4. ‘>>log’ 以追加的方式记录日志
  5. 2>&1 2: 标准异常输出 1: 标准输出 ，2>&1 两种输出都记录到 log 文件中
  6. & 后台启动方式
  7. BUILD_ID=dontKillMe 防止误杀包
